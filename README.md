# MediAgent — Multi-Agent Healthcare Diagnostic Assistant

A graduate AI project demonstrating multi-agent system design using classical AI techniques from Russell & Norvig's *Artificial Intelligence: A Modern Approach*.

## AI Techniques Demonstrated

| Technique | Agent | Chapter |
|---|---|---|
| PEAS Agent Architecture | OrchestratorAgent (utility-based) | Ch. 1-2 |
| First-Order Logic + Forward Chaining | DiagnosisAgent | Ch. 7-9 |
| Naive Bayes ML Classifier | DiagnosisAgent | Ch. 20 |
| A* Search | TreatmentPlannerAgent | Ch. 3-4 |
| Constraint Satisfaction (CSP) | DrugSafetyAgent | Ch. 5 |
| Multi-Agent Orchestration | OrchestratorAgent state machine | Ch. 1-2 |

## System Architecture

```
User symptoms
    ↓
OrchestratorAgent (state machine)
    ↓
SymptomCollectionAgent  →  normalizes symptoms
    ↓
DiagnosisAgent          →  FOL rules + Naive Bayes → top-3 diagnoses
    ↓
TreatmentPlannerAgent   →  A* search → optimal treatment path
    ↓
DrugSafetyAgent         →  CSP → drug interaction check
    ↓
ExplanationAgent        →  plain-English summary → Streamlit UI
```

## Example Interaction

**Patient input:**
> "I have a fever, body aches, fatigue, and chills."

**MediAgent output:**
```
Top diagnoses:
  1. Flu             (confidence: 0.91)  [FOL rules confirmed + NB classifier]
  2. Pneumonia       (confidence: 0.06)
  3. Bronchitis      (confidence: 0.03)

Recommended treatment pathway (A* optimal path):
  diagnosed → rest_prescribed → otc_medication → treated
  (Severity: mild — starting from rest_prescribed)

Safe medications:
  Oseltamivir, Ibuprofen, Acetaminophen
  No drug interactions detected.

Explanation:
  Your symptoms — fever, body aches, fatigue, and chills — are consistent
  with Influenza (Flu). Rest and over-the-counter medication are recommended.
  If symptoms worsen after 48 hours, seek a follow-up consultation.
```

## Setup

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the ML model (generates ml/model.pkl)
cd mediagent
python ml/train.py

# 3. Launch the app
streamlit run ui/app.py
```

## File Structure

```
mediagent/
├── agents/                        # 6 specialist agents + orchestrator
│   ├── orchestrator.py            #   OrchestratorAgent — PEAS state machine
│   ├── symptom_collection.py      #   SymptomCollectionAgent — input normalisation
│   ├── diagnosis.py               #   DiagnosisAgent — FOL + Naive Bayes
│   ├── treatment_planner.py       #   TreatmentPlannerAgent — A* search
│   ├── drug_safety.py             #   DrugSafetyAgent — CSP interaction check
│   └── explanation.py             #   ExplanationAgent — plain-English summary
│
├── knowledge/                     # Symbolic AI knowledge base
│   ├── disease_rules.py           #   Symptom vocabulary + FOL disease rules
│   ├── drug_interactions.py       #   Drug lists, forbidden pairs, alternatives
│   └── treatment_graph.py         #   Treatment-state graph + A* heuristics
│
├── ml/                            # Machine learning pipeline
│   ├── dataset_generator.py       #   Synthetic symptom dataset builder
│   ├── train.py                   #   Naive Bayes training + model serialisation
│   └── model.pkl                  #   Trained classifier (generated by train.py)
│
├── docs/                          # Project documentation
│   ├── architecture.md            #   System and agent architecture overview
│   ├── algorithms.md              #   Deep-dive on each AI algorithm used
│   ├── data_flow.md               #   End-to-end data flow diagram and notes
│   └── final_report.md            #   Full academic report + individual contributions
│
├── ui/                            # Streamlit chat interface
│   └── app.py
│
└── tests/                         # Unit tests + integration tests (60 tests)
```
